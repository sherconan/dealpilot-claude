# 智谱 AI · B+ 轮融资简介

> 来源说明：本文档基于 2024 年 1-12 月公开新闻报道、智谱开放平台官网披露、爱企查工商信息综合整理，

## 1. 公司画像（Company）

- **公司全称**：北京智谱华章科技有限公司（Zhipu AI · ZHIPU AI）
- **成立时间**：2019 年 6 月（清华 KEG 知识工程实验室技术成果产业化）
- **总部**：北京海淀
- **赛道**：通用人工智能 / 大语言模型 / 中文 AGI 基础模型 + 应用
- **产品矩阵**：
  - **GLM-4 / GLM-4-Plus** — 1300 亿参数旗舰大模型
  - **ChatGLM-3** — 6B/9B 开源版本（HuggingFace 下载累计破 1500 万）
  - **AutoGLM** — Agent 自动化（覆盖手机 / 网页 / 桌面操作）
  - **CogView** — 文生图模型
  - **CogVideoX** — 文生视频开源
  - **智谱清言**（chatglm.cn）— C 端对话产品
  - **智谱开放平台** — B 端 API 入口
- **使命**：让机器像人一样思考；构建中国自主可控的通用大模型基础设施
- **公司定位**：中国大模型"四小龙"之一（与月之暗面、百川、零一万物并列），技术派代表

## 3. 问题与机会（Problem & Why Now）

- 中国需要自主可控的中英文双语基础模型，避免国外模型出口管制风险
- ToG（政府）/ ToB（央国企）市场对国产大模型刚需，但要求技术指标对标 GPT-4
- 时间窗口：2024 是国产大模型从"模型能力比拼"向"应用场景独占 + 国产化替代"的关键年
- C 端流量入口（智谱清言）+ B 端开放平台 + ToG 政企三轮驱动

## 4. 产品与解决方案（Product / Solution）

- **GLM-4-Plus**：10 万亿 token 训练，对标 GPT-4 在中文 SuperCLUE / C-Eval / MMLU 等基准
- **AutoGLM**：业内首个开源手机 Agent（2024 Q4），可在手机端完成订外卖、抢票、预定酒店
- **CogVideoX**：文生视频开源模型，技术对标 Sora
- **API 价格**：GLM-4-Air ¥1/百万 token，GLM-4-Plus ¥50/百万 token
- **可防御性**：清华 KEG 学术起家，论文/开源生态深厚，ChatGLM 系列开源累计 1500 万下载
- **企业定制**：MaaS 模式 + 私有化部署 + 行业垂类模型（医疗 / 金融 / 法律）

## 5. 牵引（Traction · 截至 2024 Q4）

- **B 端 API 月调用量**：日均 1.4 亿 token（智谱开放平台公开数据）
- **B 端公开客户**：招商局 / 金山办公 / 合合信息 / 北京东方科脉 / 携程 / 国家电网 / 平安
- **ToG**：北京 / 上海 / 深圳市级智算中心合作；央国企大模型项目入选率 60%+
- **C 端 MAU**（智谱清言）：约 800 万（2024 Q3 估算）
- **开源生态**：ChatGLM-3 系列 HuggingFace 下载累计 1,500 万 +，GitHub Star 50K+
- **专利**：已申请大模型相关专利约 200 项

## 6. 商业模式（Business Model）

- **B 端 API**（按 token 计费，70% 营收占比）
- **企业私有化部署**（一次性 license + 年维护，20% 营收）
- **ToG 大项目**（政府智算中心 / 央国企）（10% 营收，毛利高但收款周期 9-12 月）
- **C 端 freemium**（智谱清言）目前不收费，作为 B 端流量入口
- 长期：基于 GLM-4 流量入口的 Agent 生态分润 + 企业大模型私有化

## 7. 市场（Market）

- 中国生成式 AI 市场 2025 预计 ¥1,500 亿（IDC 2024）
- 大模型应用层 2024-2027 复合增长率 87%
- ToG / 央国企国产替代市场单独规模 2025 预计 ¥500 亿
- 智谱定位：技术派 + ToG 政企标杆 + B 端开放平台
- TAM：中国 AGI 应用层 + ToG 国产替代 → 5-10 年内潜在万亿级

## 8. 竞争（Competition）

- **国内大模型四小龙**：月之暗面（C 端 / 长文本）、百川（toB / 创业老兵）、零一万物（多模态 / 李开复）
- **国内大厂**：百度文心、阿里通义、字节豆包、华为盘古
- **价格屠夫**：DeepSeek（¥1/百万 token 直接打到底）
- **国外**：OpenAI、Anthropic、Google
- **智谱差异化**：清华 KEG 学术资源 + ToG 政企关系 + 开源生态深度

## 10. 本轮 B+ 核心条款

- **本轮募资**：¥30 亿（约 $4.2 亿）
- **投后估值**：¥200 亿（约 $28 亿）
- **本基金拟出资**：¥1.5 亿（占本轮 5%）
- **轮次性质**：优先股 · Series B+
- **资金用途**：算力采购（60%）、研发团队扩充（25%）、ToG 标杆项目（15%）
- **退出路径**：港交所 18C 章特专科技规则上市 / 战略并购（阿里 / 腾讯生态） / S 基金老股转让

## 11. 风险与挑战（Risks）

- **算力依赖**：高端 GPU 受美国出口管制，长期算力供应不确定
- **价格战**：DeepSeek 把 API 价格打到 ¥1/百万 token，毛利长期承压
- **ToG 应收账款长**：政企项目收款周期 9-12 个月，现金流压力
- **技术派转商业**：唐杰团队学术氛围浓，商业化执行 vs 月之暗面的市场速度
- **监管**：生成式 AI 内容合规、网信办算法备案、数据出境

## 12. 独立风险点 / 待尽调（Diligence Hot List）

- ToG 大项目应收账款周转天数 vs 现金流储备
- 训练数据来源合规审计 + 网信办算法备案进度
- 算力采购合同条款（华为 910B / NVIDIA H100 / 寒武纪）
- 关键员工股权结构（唐杰投票权安排、清华母校占比）
- ToG 项目毛利真实情况（剔除算力成本后）
